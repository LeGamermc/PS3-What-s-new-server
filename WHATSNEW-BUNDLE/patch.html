!!Patch only functional for 4.92.2 CFW EvilNAT!! (learn more)
Due to the method used and testing it is recommended as this patch has been made on evilnat 4.92.2 CFW firmware other firmware will be unofficially supported and can cause a high risk of soft bricking that will include the need of reinstalling the firmware on your ps3 proceed with precaution

Note: please make sure to watch out this page or the xmb whats new channel to stay updated about stuff that would change
Note (bis): WebMAN is required for the procces make sure to have it downloaded and setup on your ps3

Installation Steps

  1. Download the patched explore_plugin.sprx file explore_plugin.sprx into your ps3 or usb and save it somewhere you can find it later easily
  2. Go on your ps3 and enable under [Network] -> [CFW settings] -> [File manager] -> [enable /dev_blind]
  3. (usb): Go on [Games] -> [WebMAN Games] -> [WebMAN Configuration] -> [File manager] then go to /dev_usb/explore_plugin.sprx use the copy function to copy it under /dev_blind/vsh/module/explore_plugin.sprx [make sure to remove the file before recopying it]
  4. (Hard drive): Go on [Games] -> [WebMAN Games] -> [WebMAN Configuration] -> [File manager] then go to /dev_hdd0/the_folder_you_have_downloaded_the_file_in/explore_plugin.sprx use the copy function to copy it under /dev_blind/vsh/module/explore_plugin.sprx [make sure to remove the file before recopying it]
  5. (ftp): copy the file you downloaded locally to copy it under /dev_blind/vsh/module/explore_plugin.sprx [make sure to remove the file before recopying it]
  6. Reboot your ps3
  7. Enjoy!
